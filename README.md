# 핸드팬 스케일 추천 애플리케이션

국내 사용자들의 사용 후기를 기반으로 최적의 핸드팬 스케일을 추천하는 Next.js 애플리케이션입니다.

## 기능

- 4가지 연주 목적별 스케일 추천 (입문용, 요가·명상·힐링, 밝은 Major, 개성강한분위기)
- 딩의 피치별 필터링 기능
- 전체 스케일 목록 탐색
- 스케일 상세 정보 및 영상 시청

## 기술 스택

- Next.js 14.2.0
- React 18
- TypeScript
- Tailwind CSS
- Framer Motion

## 설치 및 실행

```bash
# 의존성 설치
npm install

# 개발 서버 실행
npm run dev

# 프로덕션 빌드
npm run build
npm start
```
